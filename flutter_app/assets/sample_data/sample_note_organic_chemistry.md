# Organic Chemistry Revision

## Functional Groups

- Alcohols contain the **hydroxyl group** `-OH`.
- Carboxylic acids contain the **carboxyl group** `-COOH`.
- Aldehydes contain the **formyl group** `-CHO`.

## Acidity Trend

Carboxylic acids are more acidic than alcohols because the conjugate base is resonance-stabilized.

## Quick Comparison

| Class | Key Group | Example |
| --- | --- | --- |
| Alcohol | `-OH` | Ethanol |
| Aldehyde | `-CHO` | Ethanal |
| Ketone | `>C=O` | Propanone |
