# Data Structures Core Concepts

## Arrays

Arrays store elements in contiguous memory and support indexed access in `O(1)` time.

## Hash Maps

Hash maps use a hash function to map keys to buckets, so average lookup is `O(1)`.

## Trees

Balanced binary search trees maintain ordered data with `O(\log n)` insertion and lookup.

## Revision Prompt

When should you prefer a hash map over a tree?

- Use a hash map when fast average lookup matters more than sorted traversal.
- Use a tree when ordering, range queries, or predecessor/successor lookups matter.
